New Features for Oracle Database Release 18c

New features for Oracle Database Release 18c include new QBE item methods, new data types for QBE operator $orderby, and B-tree indexing with additional data types.

Note:

Supported features can vary among SODA implementations. The new features listed here are not necessarily supported by all implementations.

  • New item methods: $abs, $date, $size, $timestamp, and $type.

  • Use of data types date and timestamp with QBE operator $orderby.

  • B-tree indexing with date and timestamp data types.
