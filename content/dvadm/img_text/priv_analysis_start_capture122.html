Description of the illustration priv_analysis_start_capture122.png

Image priv_analysis_start_capture122.png displays the Privilege Analysis: Start Capture dialog box. From top to bottom, right to left, are the following components:

  • Policy Name label, which displays the name of the privilege analysis policy

  • Run, which provides two options: Immediate and Later

  • Date, which is activated when you select Later. It displays the current date, which you can select from a calendar

  • Following Date are the Hr, Min, and Sec menus, which also become active if you select Later, and which enable you to specify the hour, minute, and seconds for the privilege capture to begin

  • Target Time label, which displays the date and time when you selected the Start Capture command.

  • At the lower right corner, the OK and Cancel buttons
