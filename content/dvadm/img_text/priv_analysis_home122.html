Description of the illustration priv_analysis_home.png

The image priv_analysis_home.png shows the following components, from left to right, top to bottom:

  • The breadcrumb: Oracle Database, Performance, Availability, Security, Schema, Administration

  • The label Privilege Analysis

  • Text describing privilege analysis (which is covered in this guide)

  • The Policies label

  • A horizontal menu with the command Actions, View, Create Capture, Start Capture, Stop Capture, Generate Report, View Reports, Delete Capture

  • A table with these columns to describe existing privilege analysis policies: Policy, Active, Capture Scope (which has the sub-columns Type, First Start Time, Last End Time, Total Capture Duration), and Unused Privileges (which has the sub-columns System and Object)
