1 Oracle Database Installation Checklist

Use checklists to review system requirements, and to plan and carry out Oracle Database installation.

Oracle recommends that you use checklists as part of your installation planning process. Using a checklist ensures that your server hardware and configuration meets minimum requirements for this release, and enables you to carry out a successful installation.
