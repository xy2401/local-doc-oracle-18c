Changes in Oracle Database 18c, Version 18.1

This section lists changes in Oracle Database Data Cartridge Developer's Guide for Oracle Database release 18c, version 18.1.

See Oracle Database New Features Guide for a complete list of new features in this release.

Support for local domain index on reference partitioned tables is enhanced. A local domain index can now be created on a reference partitioned table whose root table is a composite partitioned table.
