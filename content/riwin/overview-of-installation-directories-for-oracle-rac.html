2.6 Overview of Installation Directories for Oracle RAC

Both an Oracle Base directory and an Oracle Home directory are used for every installation of Oracle Database software.

Additionally, on the Windows operating system, Oracle provides a home name for each Oracle Home directory.

  • Overview of Oracle Base Directories
    The Oracle base directory is the location where Oracle software and configuration files are stored.
  • Overview of Oracle Home Directories
    The Oracle home directory is located under the Oracle base directory.
