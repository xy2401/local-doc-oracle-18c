2.4 Installing Oracle RAC and Oracle RAC One Node Databases

Understand the process to install Oracle Real Application Clusters (Oracle RAC) and Oracle RAC One Node Databases.

Installation of Oracle Real Application Clusters (Oracle RAC) and Oracle RAC One Node Databases is a two-step process:

  1. Install Oracle RAC software by running the Oracle Database installer.

  2. Create and Configure Oracle RAC and Oracle RAC One Node Databases using Database Configuration Assistant (DBCA).
