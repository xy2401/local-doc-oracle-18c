6.12.3 Net Service Names Created by DBCA

Oracle Database Configuration Assistant creates net service names for connecting to the database instances.

  • Net Service Names for Database Connections
    Clients that connect to any instance of Oracle RAC use the SCAN in the connect descriptor.
  • Net Service Names for Instance Connections
    Clients that connect to a particular instance of the database use the net service name for the instance.
