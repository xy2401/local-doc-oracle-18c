3.7.3 Obtain the Password for the Oracle Home User

If you use Oracle Database Configuration Assistant to create an Oracle Home User, then create an Oracle Home User password.

Log in as the user that installed the Oracle Database software and perform the following task:
  • If an Oracle Home User was specified during installation, then obtain that user password.

See Also:

Oracle Grid Infrastructure Installation and Upgrade Guide for Microsoft Windows x64 (64-Bit) for more information about the Oracle Home User.
