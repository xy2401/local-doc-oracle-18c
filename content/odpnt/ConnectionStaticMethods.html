6.4.4 OracleConnection Static Methods

The OracleConnection static methods are listed in Table 6-31.

Table 6-31 OracleConnection Static Methods

Method Description

Equals

Inherited from System.Object (Overloaded)

ClearPool

Clears the connection pool that is associated with the provided OracleConnection object.

Not supported in a .NET stored procedure

ClearAllPools

Clears all connections from all the connection pools

Not supported in a .NET stored procedure
