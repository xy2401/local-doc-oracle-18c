Description of the illustration dbr_process_locate_wkload.gif

The upper left portion of the page provides values for the Database, Version, Replay Name, and Logged In As fields.

The upper right portion of the page states "Step 1 of 5," a Cancel button appears to the left of this text, and a Next button appears to the right of this text.

The lower half of the page contains the following text, followed by two options: "The captured workload directories must be accessible from this database."

The first option is: "Copy the workload directories to this host from another host." This option is selected.

Thesecond option is: "Use an existing directory with multiple workload subdirectories on this host." This option is not selected.
