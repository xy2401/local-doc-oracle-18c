Verifying Operating System Patches on Oracle Solaris 10

To check if your operating system has the required Oracle Solaris 10 patches for installation, run the following command:

  1. To determine whether an operating system patch is installed and whether it is the correct version:
    # /usr/sbin/patchadd -p | grep patch_number
    For example, to determine if any version of the 119963 patch is installed:
    # /usr/sbin/patchadd -p | grep 119963

Note:

  • Your system may have more recent versions of the listed patches installed. If a listed patch is not installed, then determine if a more recent version is installed before installing the version listed.

  • If an operating system patch is not installed, then download and install it from My Oracle Support.
