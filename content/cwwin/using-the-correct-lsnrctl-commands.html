10.3.4 Using the Correct LSNRCTL Commands

Do not attempt to use the lsnrctl programs from Oracle home locations for earlier releases because they cannot be used with the new release.

  • Use the Listener Control utility, lsnrctl, located in the Oracle Grid Infrastructure 18c home to administer local and SCAN listeners for Oracle Clusterware and Oracle ASM 11g Release 2.
