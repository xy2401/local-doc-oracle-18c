3.8 Oracle Enterprise Manager Requirements

Verify that your installed Oracle Enterprise Manager meets the minimum requirements for use with Oracle Grid Infrastructure.

All Oracle Enterprise Manager products that you use on your system must be of the same release. Oracle Database 12c Release 1 (12.1) and later releases do not support releases of Enterprise Manager earlier than Oracle Enterprise Manager Cloud Control 12c.
