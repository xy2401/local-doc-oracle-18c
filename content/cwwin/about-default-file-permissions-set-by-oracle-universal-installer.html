9.4 About Default File Permissions Set by Oracle Universal Installer

Oracle Grid Infrastructure is installed in the ORACLE_HOME directory, by default. Oracle Universal Installer sets the permissions for this directory, and for all files and directories under this directory.

For the ORACLE_HOME of Oracle Grid Infrastructure, the following permissions are granted:

  • Full control - Administrators, SYSTEM, ORA_GRID_LISTENERS, Oracle Installation User, Oracle Home User

  • Read, Execute, and List Contents - Authenticated Users
