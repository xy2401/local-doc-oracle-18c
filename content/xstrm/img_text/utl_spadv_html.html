Description of the illustration utl_spadv_html.gif

The illustration shows part of the main page of an HTML report generated by the SHOW_STATS_HTML procedure.

The report includes the following links in the "Contents" section:

  • Legend

  • Event Metrics

  • Paths

  • Path Summary

  • Path Level Event Summary

  • Rate Level Stats

The "Legend" section contains a table with the following columns: "Acronym," "Component," "Throughput," and "Rate 2." The table includes the following rows:

  • CAPTURE, Capture Component, msgs enqueued/sec, msgs captured/sec

  • LMR, Log Miner Reader, n/a, n/a

  • LMP, Log Miner Preparer, n/a, n/a

  • LMB, Log Miner Builder, n/a, n/a

  • CP, Capture Process, n/a, n/a

  • Q, Queue, enqueued/sec, no of msgs in queue

  • PS, Propagation Sender, msgs/sec, bytes/sec

  • PR, Propagation Receiver, n/a, n/a

  • APPLY, Apply component, msgs applied/sec, txns applied/sec

  • APR, Apply Reader, n/a, n/a

  • APC, Apply Coordinator, n/a, n/a

  • APS, Apply Slave, n/a, n/a

The "Event Metrics" section contains a table with the following columns: "Metric" and "Description." The table includes the following rows:

  • IDLE%, Percent of time in the run, spent waiting on upstream component

  • FLWCTRL%, Percent of time in the run, spent waiting on downstream component

  • TOPEVENT, Non-idle, Non-flwctrl Event which occupies most of run time

  • TOPEVENT%, Percent of time in the run, spent on Topevent

The "Paths" section contains a "Path 1" table with the following columns: "Component," "Name," and "Database." The table includes the following rows:

  • CAPTURE, CAP$_XOUT_1, blank

  • Q, "XSTRMADMIN"."Q$_XOUT_2", blank

  • APPLY, XOUT, blank

The "Path Level Summary" section contains a table with the following columns: "Path Id," "Avg CAPTURE msgs/sec," "Avg CAPTURE latency," "Avg APPLY txns/sec," "Avg APPLY msgs/sec," and "Avg APPLY latency." The table includes the following row:

  • 1, 9464.64, 1.25, 85.37, 9221.91, 1.56
