Understanding Cluster Configuration Options

Review these topics to understand the cluster configuration options available in Oracle Grid Infrastructure 18c.

  • About Oracle Standalone Clusters
    An Oracle Standalone Cluster hosts all Oracle Grid Infrastructure services and Oracle ASM locally and requires direct access to shared storage.
  • About Oracle Extended Clusters
    An Oracle Extended Cluster consists of nodes that are located in multiple locations called sites.
