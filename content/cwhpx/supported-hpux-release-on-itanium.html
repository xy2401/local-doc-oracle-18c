Supported HP-UX Release on Itanium

Use the following information to check the minimum supported HP-UX Itanium distribution:

Table 3-1 HP-UX Itanium Minimum Operating System Requirements

Item Requirements

HP-UX 11.31 operating system

HP-UX 11.31 September 2014 B.11.31.1409 or later.

Oracle Clusterware

  • HP Serviceguard A.11.20

  • HP Serviceguard A.11.20 extension for RAC

Note:

HP Serviceguard is optional. It is required only if you want to use shared logical volumes for Oracle Clusterware or database files.
