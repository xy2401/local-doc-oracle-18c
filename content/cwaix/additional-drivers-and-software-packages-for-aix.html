Additional Drivers and Software Packages for IBM AIX on POWER Systems (64-Bit)

Information about optional drivers and software packages.

You are not required to install additional drivers and packages, but you may choose to install or configure these drivers and packages.

  • Installing Oracle Messaging Gateway
    Oracle Messaging Gateway is installed with Enterprise Edition of Oracle Database. However, you may require a CSD or Fix Packs.
  • Installation Requirements for ODBC and LDAP
    Review these topics to install Open Database Connectivity (ODBC) and Lightweight Directory Access Protocol (LDAP).

  • Installation Requirements for Web Browsers
    Web browsers are required only if you intend to use Oracle Enterprise Manager Database Express and Oracle Enterprise Manager Cloud Control. Web browsers must support JavaScript, and the HTML 4.0 and CSS 1.0 standards.
