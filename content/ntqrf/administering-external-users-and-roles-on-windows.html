11 Administering External Users and Roles on Windows

External users and roles are in general defined by something external to Oracle Database.

In a Windows environment, they are defined by the operating system.

Learn about the external user and external role creation and management using a combination of Oracle Database command-line tools, Registry Editor, and other Windows tools.

  • Overview of Manually Administering External Users and Roles
    Manually configure administrators, operators, users, and roles to be authenticated by the operating system.
