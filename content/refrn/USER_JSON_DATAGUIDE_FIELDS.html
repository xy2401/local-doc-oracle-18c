6.489 USER_JSON_DATAGUIDE_FIELDS

USER_JSON_DATAGUIDE_FIELDS extracts the path and type information from all the data guides in the current user’s schema, which are the data guides returned to the user by the USER_JSON_DATAGUIDE view. Its columns (except for OWNER) are the same as those in ALL_JSON_DATAGUIDE_FIELDS.

Note:

This view is available starting with Oracle Database release 18c, version 18.1.
