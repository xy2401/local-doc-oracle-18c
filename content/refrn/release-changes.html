Deprecated Features

  • The FILE_MAPPING initialization parameter has been deprecated.

    See "FILE_MAPPING" for more information.

  • The ALL_SCHEDULER_CREDENTIALS view has been deprecated.

    See "ALL_SCHEDULER_CREDENTIALS" for information about alternatives.

  • The DBA_NETWORK_ACL_PRIVILEGES view has been deprecated.

    See "DBA_NETWORK_ACL_PRIVILEGES" for information about alternatives.

    See "DBA_NETWORK_ACLS" for information about alternatives.

    See "DBA_SCHEDULER_CREDENTIALS" for information about alternatives.

    See "USER_NETWORK_ACL_PRIVILEGES" for information about alternatives.

  • The USER_SCHEDULER_CREDENTIALS view has been deprecated.

    See "USER_SCHEDULER_CREDENTIALS" for information about alternatives.

    See "V$OBJECT_USAGE" for information about alternatives.
