1.39 CLUSTER_DATABASE

CLUSTER_DATABASE is an Oracle RAC parameter that specifies whether Oracle RAC is enabled.

Property Description

Parameter type

Boolean

Default value

false

Modifiable

No

Modifiable in a PDB

No

Range of values

true | false

Basic

Yes

Oracle RAC

For all instances, the value must be set to true.
