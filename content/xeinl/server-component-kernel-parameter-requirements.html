3.3 Server Component Kernel Parameter Requirements

The Oracle Database XE preinstallation RPM checks your system for the following kernel parameter settings.

If the kernel parameter values of your system are less than the values listed in Table 3-2, then the preinstallation RPM sets the recommended minimum kernel parameter values for you.

The values set in the /etc/sysctl.d/97-oracle-database-sysctl.conf file persist on system restarts.
