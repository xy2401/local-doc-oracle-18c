Oracle Trace File Analyzer Service classifies each trace file and provides an interface to browse files by various classifications. This makes it easy to find important files across thousands of trace files generated by various components.

The interface shows a tree navigation menu on left to browse files by type, or hostname or by actual directory or by TFA collections. You can also filter the file list by cluster, node or TFA collection. There is a "search" option to find file names by keyword search.

Use the tree on the left to click browse files by file type, folder, and collection.

Click a file to open it or merge multiple files to view combined events across multiple files.
