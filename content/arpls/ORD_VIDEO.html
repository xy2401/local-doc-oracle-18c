222 ORD_VIDEO

This Oracle Multimedia package supports the management and manipulation of video data stored in BLOBs and BFILEs.

Note:

ORD_VIDEO is deprecated in Oracle Database Release 18c. Oracle recommends that you do not use deprecated features. Support for deprecated features is for backward compatibility only.

The Oracle Multimedia ORD_VIDEO package provides functions and procedures to perform common operations such as exporting video data from Oracle Database into operating system files, and extracting information from video data. Information from video data such as format, width, height, and frame rate can be extracted from video data into an XML string and optionally into individual attributes.

For a complete description of this package within the context of Oracle Multimedia, see ORD_VIDEO in the Oracle Multimedia Reference.
