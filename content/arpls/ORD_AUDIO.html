217 ORD_AUDIO

This Oracle Multimedia package supports the management and manipulation of audio data stored in BLOBs and BFILEs.

Note:

ORD_AUDIO is deprecated in Oracle Database Release 18c. Oracle recommends that you do not use deprecated features. Support for deprecated features is for backward compatibility only.

The Oracle Multimedia ORD_AUDIO package provides functions and procedures to perform common operations such as exporting audio data from Oracle Database into operating system files, and extracting information from audio data. Information from audio data such as format, encoding type, and sampling rate can be extracted from audio data into an XML string and optionally into individual attributes.

For a complete description of this package within the context of Oracle Multimedia, see ORD_AUDIO in the Oracle Multimedia Reference.
