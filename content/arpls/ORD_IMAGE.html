221 ORD_IMAGE

This Oracle Multimedia package supports the management and manipulation of image data stored in BLOBs and BFILEs.

Note:

ORD_IMAGE is deprecated in Oracle Database Release 18c. Oracle recommends that you do not use deprecated features. Support for deprecated features is for backward compatibility only.

The Oracle Multimedia ORD_IMAGE package provides functions and procedures to perform common operations such as exporting image data from Oracle Database into operating system files, creating thumbnail images, converting images from one format to another, and extracting information from image data. Information from image data such as format, width, height, and compression type can be extracted from image data into an XML string and optionally into individual attributes.

For a complete description of this package within the context of Oracle Multimedia, see ORD_IMAGE in the Oracle Multimedia Reference.
